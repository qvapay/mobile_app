# QvaPay Mobile App

**QvaPay** is a decentralized fintech platform that offers a non-custodial wallet, P2P marketplace, crypto payment gateway, and seamless integration with real-world financial services — all in one mobile app.

> “Building financial technologies that are free and accessible for everyone.”

## 🚀 About QvaPay

QvaPay enables individuals and businesses—especially in underbanked regions like Latin America and the Caribbean—to access the global financial system through crypto and digital balance infrastructure. With QvaPay, users can:

- Manage a digital USD balance
- Send and receive remittances
- Trade via a decentralized P2P market
- Deposit and withdraw via crypto, bank transfers, or third-party rails
- Access e-commerce and merchant tools
- Interact with APIs for custom integrations

## 📱 About This App

This mobile application is a secure and fast gateway to your QvaPay account. Developed in **React Native**, the app provides a fluid experience to manage your finances on the go.

### ✨ Core Features

- ✅ Non-custodial crypto wallet
- ✅ USD-equivalent digital balance
- ✅ P2P marketplace for buying/selling funds
- ✅ Real-time deposits and withdrawals
- ✅ KYC and security features (2FA, biometric auth)
- ✅ Transaction history and receipts
- ✅ API integrations for businesses

## ⚙️ Tech Stack

- **React Native** (cross-platform)
- **Axios for API communication**
- **react-native-keychain** for secure token storage
- Backend: [QvaPay API](https://qvapay.com/docs)

## 🧠 Architecture Overview

The app is structured with modular screens, secure session handling, and API-first communication. The backend handles authentication, transactions, KYC verification, and crypto operations.

> 📦 Authentication is based on access tokens stored securely on-device and refreshed when needed.

## 🛠 Getting Started (for developers)

To build and run the app locally:

```bash
git clone https://github.com/qvapay/mobile-app.git
cd mobile-app
yarn install
yarn android # or yarn ios
```

> Requires Node.js, Yarn, and either Android Studio or Xcode installed.
